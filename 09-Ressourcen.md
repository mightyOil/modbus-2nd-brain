# Ressourcen & Weiterführendes

## Offizielle Quellen (unbedingt lesen!)
- **Modbus Organization**: https://www.modbus.org/
- **Protocol Specification V1.1b3** (PDF): https://www.modbus.org/file/secure/modbusprotocolspecification.pdf
- **Modbus Serial Line Protocol V1.02**: Für RTU/ASCII Implementierungen
- **Modbus TCP/IP Implementation Guide**: https://www.modbus.org/

## Libraries & Tools
- **pymodbus** (Python): https://github.com/pymodbus-dev/pymodbus
- **Modbus Tools** (GUI, Test-Tools): https://www.modbustools.com/
- **QModMaster** (Open Source GUI)
- **Modbus Poll / Slave** (kommerziell, sehr gut zum Testen)
