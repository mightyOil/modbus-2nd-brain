# MODBUS - Tiefes Verständnis | 2nd Brain Projekt

**Erstes Projekt im Obsidian 2nd Brain**

Willkommen zu deinem persönlichen Lernprojekt für ein tiefes Verständnis des MODBUS-Protokolls! 

Dieses Repository ist als Obsidian-Vault optimiert und mit GitHub synchronisiert – so hast du überall Zugriff (Desktop, Mobile, Tablet). 

## Projektziele
- Fundiertes technisches Verständnis von MODBUS (nicht nur Oberfläche)
- Strukturierte, atomare Notes im Zettelkasten-Stil
- Praktische Beispiele (Python mit pymodbus)
- Ressourcen, Best Practices und nächste Schritte

## Wie du das Vault nutzt
1. Clone oder sync dieses Repo in Obsidian (via Git-Plugin oder Obsidian Git)
2. Öffne den Ordner als Vault
3. Starte mit [[MOC-Modbus-Index]]
4. Folge den Links zu den Themen
5. Ergänze eigene Notizen, Tags (#modbus #industrial #automation)

## Struktur
- `MOC-*.md`: Maps of Content (Einstiegspunkte)
- `XX-Thema.md`: Atomare, verlinkte Notes
- `Examples/`: Code-Beispiele
- `Resources/`: Weiterführende Links & PDFs
